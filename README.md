# excel-to-sql

```
Usage of excel-to-sql:

/* ja */
Excelファイル(.xlsx)を読み込んでInsert文を出力します

Options
  -c string
        Please Input Columns
  -columns string
        Please Input Columns
  -f string
        *Required* Please Input FilePath
  -file string
        *Required* Please Input FilePath
  -n int
        *Required* Please Input Sheet Number( 0 start ) (default -1)
  -num int
        *Required* Please Input Sheet Number( 0 start ) (default -1)
  -o string
        Please Input Output (default "output.sql")
  -output string
        Please Input Output (default "output.sql")
  -s int
        Please Input Separator Number (default 1)
  -separator int
        Please Input Separator Number (default 1)
  -t string
        *Required* Please Input Table
  -table string
        *Required* Please Input Table
```
